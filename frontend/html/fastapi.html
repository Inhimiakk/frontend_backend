Учебный материал

FastAPI — современный фреймворк для создания веб-API на Python. Он автоматически генерирует документацию Swagger и валидирует данные через Pydantic.

  • Маршрут — функция с декоратором @app.get, @app.post и т.д.
  • Pydantic-модель описывает входные данные (типизация + проверка).
  • Сервер запускают через uvicorn main:app --reload.
from fastapi import FastAPI
from pydantic import BaseModel

app = FastAPI()

class Item(BaseModel):
    name: str
    price: float

@app.post(\"/items\")
def create_item(item: Item):
    return {\"ok\": True, \"item\": item}

Сделай своё API

Напиши FastAPI-приложение с ресурсом /notes:

  • GET /notes — вернуть список заметок (пусть пока пустой/заглушка);
  • POST /notes — принять JSON с полями title, text и вернуть эту заметку обратно.

Добавь Pydantic-модель для заметки.
