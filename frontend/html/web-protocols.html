Учебный материал

HTTP — протокол обмена между клиентом (браузер) и сервером. Клиент отправляет запрос, сервер отвечает.

  • Методы: GET (получить), POST (создать), PUT/PATCH (обновить), DELETE (удалить).
  • Коды: 200 OK, 201 Created, 400 Bad Request, 401 Unauthorized, 404 Not Found, 500 Server Error.
  • HTTPS — HTTP поверх шифрования TLS (безопасный канал).
  • REST — стиль проектирования API: ресурсы + методы + JSON.

Пример запроса:

GET /courses HTTP/1.1
Host: example.com
