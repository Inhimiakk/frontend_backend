CSS

Стилизация, каскад, Flexbox и Grid.

Учебный материал

CSS (Cascading Style Sheets) — язык стилей, который задаёт внешний вид HTML-элементов: цвета, отступы, сетки, шрифты, адаптивность.

  • Селекторы выбирают элементы: по тегу (p), классу (.card), id (#header).
  • Каскад: если правил несколько, выигрывает более специфичное или то, что ниже.
  • Box-model: у каждого блока есть content → padding → border → margin.
  • Flexbox удобен для строк/колонок, выравнивания элементов.
  • Grid — для сложных таблиц/сеток.
.box{
  width: 200px;
  padding: 12px;
  margin: 10px;
  border: 1px solid #ccc;
  display: flex;
  justify-content: center;
}

Стилизуй блоки

Создай страницу с 3 карточками (div). Для каждой:

  • фон светлый/тёмный, скругление 12–16px;
  • тень;
  • отступы внутри 16px;
  • карточки расположены в ряд через Flexbox, на мобилке — колонкой.
