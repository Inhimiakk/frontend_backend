HTML

Структура страницы, семантика, формы и ссылки.

Учебный материал

HTML (HyperText Markup Language) — язык разметки, который описывает структуру веб-страницы. Он не задаёт внешний вид напрямую, а лишь определяет элементы: заголовки, абзацы, списки, изображения, формы.

Основные идеи:

  • Документ начинается с <!DOCTYPE html> и корневого тега <html>.
  • В <head> лежат метаданные: кодировка, заголовок страницы, подключение CSS/JS.
  • В <body> — видимая часть страницы.
  • Семантические теги (<header>, <main>, <section>, <footer>) помогают поиску и доступности.
  • Формы (<form>) собирают данные пользователя через <input>, <textarea>, <select>.

Пример базовой структуры:

<!DOCTYPE html>
<html lang="ru">
  <head>
    <meta charset="UTF-8">
    <title>Моя страница</title>
  </head>
  <body>
    <h1>Привет!</h1>
    <p>Это абзац текста.</p>
  </body>
</html>
Задача

Сверстай мини-страницу

Сделай HTML-страницу «О себе», где будут:

  • заголовок 1 уровня с именем;
  • 2 абзаца текста (кто ты и чем занимаешься);
  • список из 3 любимых занятий;
  • ссылка на соцсеть/портфолио;
  • картинка (можно любая из интернета).

Используй семантические теги: <header>, <main>, <footer>.
