Курс «Веб-програмування»

Повна програма курсу, теми, результати навчання та формат.

Про курс

Курс «Веб-програмування» дає системне розуміння створення сучасних веб-сайтів і веб-додатків: від верстки до взаємодії з сервером та розгортання.

  • Тривалість: 10–12 тижнів
  • Формат: лекції + практичні + домашні завдання
  • Рівень: початковий → середній
  • Результат: фінальний проєкт (портфоліо-сайт або веб-сервіс)

Модуль 1. Основи HTML

  • Структура HTML-документа, DOCTYPE
  • Семантичні теги: header, main, section, footer
  • Текстові елементи, списки, таблиці
  • Посилання, зображення, медіа
  • Форми та валідація HTML5
  • Доступність (ARIA-атрибути)

Модуль 2. CSS та макетування

  • Селектори, специфічність і каскад
  • Box-model: margin/padding/border
  • Flexbox для побудови інтерфейсів
  • CSS Grid для складних сіток
  • Адаптивність: media queries, fluid-layout
  • Анімації та transition

Модуль 3. JavaScript (Frontend)

  • Змінні, типи даних, оператори
  • Функції, об’єкти, масиви
  • DOM-модель і події
  • Форми, валідація, інтерактивність
  • Fetch API, робота з JSON
  • LocalStorage / SessionStorage

Модуль 4. Web-протоколи та мережа

  • HTTP/HTTPS: запити та відповіді
  • Методи GET/POST/PUT/PATCH/DELETE
  • Коди статусів 2xx/4xx/5xx
  • REST-архітектура
  • CORS та безпека запитів

Модуль 5. Backend на FastAPI

  • Створення API в FastAPI
  • Pydantic-моделі і валідація даних
  • Маршрути, залежності, middleware
  • Робота з базою (SQLite/PostgreSQL — базово)
  • Надсилання листів через SMTP
  • Swagger документація

Модуль 6. Сокети та реальний час

  • TCP/UDP основи
  • WebSocket для чатів/повідомлень
  • Клієнт-сервер взаємодія в реальному часі

Фінальний проєкт

Наприкінці курсу студент виконує проєкт: лендинг / портфоліо / веб-сервіс із frontend + backend. Проєкт захищається з демонстрацією коду та результату.

Що ви вмітимете
  • Верстати адаптивні сторінки на HTML/CSS
  • Додавати інтерактивність на JavaScript
  • Створювати REST API на FastAPI
  • Надсилати дані з фронтенду на сервер
  • Розгортати проєкт на хостингу
