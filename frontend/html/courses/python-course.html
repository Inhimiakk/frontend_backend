Про курс

Python — універсальна мова програмування, яка підійде і для вебу, і для аналізу даних, і для автоматизації. На курсі ви навчитеся писати чистий код та розв’язувати практичні задачі.

Що ви вивчите

  • Типи даних, змінні, оператори
  • Умови, цикли, функції
  • Списки, словники, множини
  • Файли, модулі, пакети
  • ООП: класи, наслідування, методи
  • Знайомство з бібліотеками (requests, pandas — базово)

Програма (коротко)

  • Тиждень 1–2: синтаксис, типи, базові конструкції
  • Тиждень 3–4: функції, колекції, робота з файлами
  • Тиждень 5–6: ООП і проєктування
  • Тиждень 7–8: практичні міні-проєкти
