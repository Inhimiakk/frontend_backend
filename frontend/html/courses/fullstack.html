Про курс

Повний шлях веб-розробника: від верстки та JavaScript до бекенду на Python (FastAPI) та взаємодії з базами даних. Після курсу ви зможете робити повноцінні веб-проєкти.

Що ви вивчите

  • Frontend: HTML, CSS, адаптивність, анімації
  • JavaScript: DOM, події, Fetch API, JSON
  • Мережа: HTTP/HTTPS, REST, CORS
  • Backend: Python, FastAPI, маршрути, Pydantic
  • Бази даних: CRUD, SQLite/PostgreSQL (основи)
  • Деплой: як викласти сайт/бекенд на хостинг

Програма (коротко)

  • Модуль 1: HTML + CSS (верстка, адаптивність)
  • Модуль 2: JavaScript для інтерактивності
  • Модуль 3: WEB-протоколи, API, REST
  • Модуль 4: Python основи для backend
  • Модуль 5: FastAPI + робота з даними
  • Модуль 6: фінальний fullstack-проєкт
