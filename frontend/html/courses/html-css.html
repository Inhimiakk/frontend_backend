Про курс

Цей курс — ідеальний старт для тих, хто хоче навчитися створювати веб-сторінки. Ви освоїте базову та семантичну верстку, стилізацію, адаптивність і сучасні підходи до макетування.

Що ви вивчите

  • Структуру HTML-документа, семантику, форми
  • Селектори CSS, каскад та специфічність
  • Box-model, позиціювання, псевдокласи
  • Flexbox та Grid для побудови макетів
  • Адаптивну верстку (media queries)
  • Базові анімації та ефекти

Програма (коротко)

  • Тиждень 1–2: HTML основи, теги, форми, семантика
  • Тиждень 3–4: CSS, box-model, позиціювання
  • Тиждень 5–6: Flexbox, адаптивність
  • Тиждень 7–8: Grid, UI-патерни, анімації
  • Фінал: створення власного сайту-портфоліо
